# LeadPhoneFinder

Telegram-бот для автоматического поиска B2B лидов и обогащения их контактными телефонами ЛПР (лиц, принимающих решения).

## Что делает

LeadPhoneFinder решает главную проблему B2B продаж — **получение прямого контакта с владельцем/руководителем бизнеса**.

```
Входные данные                    Результат
─────────────                    ─────────
"рестораны Сочи"         →       Excel с 150+ компаниями
                                  + телефоны ЛПР
                                  + ИНН, адреса, рейтинги

companies.xlsx (с ИНН)   →       Тот же файл
                                  + колонка "Телефон ЛПР"
```

## Возможности

| Функция | Описание |
|---------|----------|
| **Поиск компаний** | Запрос "рестораны Сочи" → парсинг 2ГИС и Яндекс.Карт |
| **Загрузка Excel** | Файл с ИНН → обогащение телефонами ЛПР |
| **Исправление опечаток** | "рестараны Сачи" → автоисправление |
| **Дедупликация** | Удаление дублей из разных источников |
| **Поиск ИНН** | По названию компании (DaData/ЕГРЮЛ) |
| **Экспорт** | Результаты в Excel с полными данными |

## Быстрый старт

### 1. Установка

```bash
git clone <repository-url>
cd LeadPhoneFinder

python -m venv venv
venv\Scripts\activate  # Windows
# source venv/bin/activate  # Linux/MacOS

pip install -r requirements.txt
playwright install chromium
```

### 2. Настройка

Скопируйте `.env.example` в `.env` и заполните:

| Параметр | Описание | Где получить |
|----------|----------|--------------|
| `BOT_TOKEN` | Токен Telegram бота | [@BotFather](https://t.me/BotFather) |
| `TELETHON_API_ID` | API ID для Telethon | [my.telegram.org](https://my.telegram.org) |
| `TELETHON_API_HASH` | API Hash для Telethon | [my.telegram.org](https://my.telegram.org) |
| `TELETHON_PHONE` | Номер телефона | Ваш номер |
| `DADATA_TOKEN` | Токен DaData (опционально) | [dadata.ru](https://dadata.ru) |

### 3. Запуск

```bash
python -m bot.main
```

## Использование

### Поиск компаний

```
/scrape → "рестораны Сочи" → Выбор источников → Обогащение → Excel
```

1. Отправьте `/scrape`
2. Введите запрос: `рестораны Сочи`
3. Выберите источники: 2ГИС, Яндекс или оба
4. Дождитесь парсинга (~2-3 минуты на 100 компаний)
5. Выберите: обогатить телефонами или скачать как есть
6. Получите Excel с результатами

### Загрузка Excel

```
[файл.xlsx] → Валидация → Обогащение → Excel с телефонами
```

1. Отправьте Excel файл с колонками "ИНН" и "Название"
2. Бот проверит ИНН и покажет статистику
3. Нажмите "Обогатить телефонами"
4. Получите обогащённый файл

## Себестоимость

| Компонент | Стоимость |
|-----------|-----------|
| VPS сервер | 500-1,500 ₽/мес |
| Парсинг 2ГИС/Яндекс | бесплатно |
| Поиск ИНН (DaData) | 0.10 ₽/запрос |
| Телефоны ЛПР (Sherlock) | бесплатно |
| **Себестоимость лида** | **~0.10-0.50 ₽** |

При продаже лидов по 100 ₽ — маржа **99%+**.

## Структура проекта

├── bot/
│   ├── main.py                 # Точка входа
