# План развития LeadPhoneFinder

---

## Текущий функционал

LeadPhoneFinder — B2B платформа для генерации лидов и автоматизированных продаж через Telegram:

1. **Поиск компаний** — парсинг 2ГИС и Яндекс.Карт по категории/городу
2. **Обогащение данных** — поиск телефонов ЛПР через Sherlock OSINT (по ИНН, ФИО директора, обратный поиск)
3. **AI-продажник** — автоматическая холодная рассылка через реальные Telegram-аккаунты с AI-ответами (GPT-4o-mini)
4. **Загрузка файлов** — импорт Excel/CSV для обогащения
5. **Мульти-аккаунт** — пул Telethon-аккаунтов с round-robin распределением

---

## 3 лучших варианта использования (сейчас)

### 1. Полный цикл: поиск → обогащение → продажа
Пользователь ищет "рестораны Сочи" → получает 150 компаний → обогащает телефонами ЛПР → запускает AI-рассылку с оффером. **Весь цикл от поиска до первого диалога — без ручной работы.**

### 2. Обогащение существующей базы
Компания с CRM выгружает Excel с ИНН → загружает в бота → получает телефоны директоров/владельцев, которых нет в открытых источниках. **Ценность: доступ к прямым номерам ЛПР, а не ресепшенам.**

### 3. Автопилот холодных продаж
Маленький бизнес (маркетинговое агентство, b2b-сервис) загружает базу контактов → AI ведёт переписку, обрабатывает возражения, назначает звонки, перенаправляет к менеджеру при "тёплом" статусе. **Ценность: 1 человек заменяет 3-5 SDR.**

## Монетизация

### Модель 1: SaaS-подписка (рекомендуемая)
| Тариф | Цена/мес | Лимиты |
|-------|----------|--------|
| Free | 0 ₽ | 10 обогащений, 5 AI-диалогов, 1 аккаунт |
| Starter | 2 990 ₽ | 200 обогащений, 50 AI-диалогов, 2 аккаунта |
| Pro | 7 990 ₽ | 1000 обогащений, 200 AI-диалогов, 5 аккаунтов |
| Business | 19 990 ₽ | Безлимит обогащений, 1000 AI-диалогов, 20 аккаунтов |

**Что считать единицей:**
- 1 обогащение = 1 компания с найденным телефоном
- 1 AI-диалог = 1 получатель в рассылке (любое кол-во сообщений)

### Модель 2: Pay-per-use (дополнительно)
- Обогащение: 15 ₽ / найденный телефон
- AI-диалог: 30 ₽ / получатель
- Поиск компаний: бесплатно (привлечение)

### Модель 3: White-label
Продажа решения маркетинговым агентствам под их брендом: 50 000-150 000 ₽ за установку + ежемесячная лицензия.

## План доработок

### Фаза 1: Стабилизация (1-2 недели)

| # | Задача | Приоритет | Файлы |
|---|--------|-----------|-------|
| 1.1 | **Биллинг и лимиты** — счётчик обогащений и диалогов на пользователя, проверка перед каждой операцией | Критический | Новый `bot/services/billing.py`, `bot/models/user.py` |
| 1.2 | **Базу данных** — перенести хранение с JSON-файлов на SQLite/PostgreSQL (кампании, пользователи, статистика) | Критический | Новый `bot/db/`, миграция `outreach_storage.py` |
| 1.3 | **Логирование ошибок** — Sentry или аналог для мониторинга в проде | Высокий | `bot/main.py`, настройка |
| 1.4 | **Graceful shutdown** — корректное завершение активных AI-вызовов при рестарте | Средний | `bot/main.py`, `bot/services/outreach.py` |
| 1.5 | **Тесты** — покрытие критических путей (AI ответы, debounce, referral) | Средний | `tests/` |

### Фаза 2: Рост ценности продукта (2-4 недели)

| # | Задача | Приоритет | Описание |
|---|--------|-----------|----------|
| 2.1 | **Аналитика кампаний** — дашборд: конверсия по статусам, средний response rate, лучшие офферы | Высокий | Новый хэндлер + сообщения |
| 2.2 | **A/B тестирование офферов** — запуск кампании с 2-3 вариантами первого сообщения, автоматический выбор лучшего | Высокий | `outreach.py` |
| 2.3 | **CRM-интеграция** — webhook при тёплом лиде (AmoCRM, Bitrix24) | Высокий | Новый `bot/services/crm_webhook.py` |
| 2.4 | **Скоринг лидов** — AI оценивает потенциал лида по ответам (1-10), сортировка | Средний | `ai_sales.py` |
| 2.5 | **Персонализация по нише** — автоматическая адаптация промпта под категорию компании (HoReCa, медицина, авто) | Средний | `ai_sales.py` |
| 2.6 | **Расширенные источники** — Google Maps, Авито, HeadHunter (компании которые нанимают = растущие) | Средний | Новые скрапперы |

| # | Задача | Описание |
