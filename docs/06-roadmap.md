## Ближайший этап: Оптимизация

**Цель:** Увеличить надёжность и скорость при больших объёмах

### Задачи

| Задача | Приоритет | Сложность | Описание |
|--------|-----------|-----------|----------|
| Кэширование результатов | Высокий | Средняя | Redis/файловый кэш для повторных запросов |
| Пул прокси | Высокий | Средняя | Ротация IP для антибана |
| История запросов | Средний | Низкая | Сохранение истории пользователя |
| Статистика партнёров | Средний | Средняя | Кто сколько лидов получил |
| Очередь задач | Низкий | Высокая | Celery/RQ для фоновой обработки |

### Детали реализации

#### Кэширование

```python
# Redis кэш для результатов парсинга
cache_key = f"scrape:{city}:{category}"
cached = await redis.get(cache_key)
if cached and cache_age < 24h:
    return cached
```

#### Пул прокси

```python
PROXIES = [
    "http://proxy1:8080",
    "http://proxy2:8080",
    ...
]

async def get_proxy():
    return random.choice(PROXIES)
```

## Следующий этап: AI Outreach

**Цель:** Автоматическая коммуникация с лидами через Telegram/WhatsApp

### Концепция

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│    Лиды     │ ──▶ │  AI-диалог  │ ──▶ │   Аудит     │ ──▶ │  Менеджер   │
│  (телефоны) │     │ (GPT fine-  │     │ (бесплатный)│     │  (закрытие) │
│             │     │   tuned)    │     │             │     │             │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
```

### Компоненты

| Компонент | Описание | Технология |
|-----------|----------|------------|
| Campaign Manager | Управление рассылками | Python + DB |
| Conversation Engine | State machine диалога | FSM |
| AI Engine | Генерация ответов | GPT-4o-mini fine-tuned |
| Objection Handler | Обработка возражений | Промпты + fine-tuning |
| Multi-channel Sender | Отправка в TG/WA | Telethon/WhatsApp API |

### Обработка возражений

| Возражение | Ответ AI |
|------------|----------|
| "Нет времени" | "Понимаю! Именно поэтому предлагаю 15-минутный аудит, который сэкономит вам часы..." |
| "Не интересно" | "Уважаю ваше мнение. Могу уточнить, что именно вас не заинтересовало?..." |
| "Дорого" | "Бесплатный аудит ни к чему не обязывает. Вы увидите точки роста..." |
| "Уже есть" | "Отлично! Как давно проводили аудит? Рынок меняется каждые 3 месяца..." |

### Требования для fine-tuning

1. **Датасет:** 200-500 реальных диалогов менеджеров
2. **Формат:** JSONL с system/user/assistant
3. **Стоимость:** ~$5-10 на обучение GPT-4o-mini
4. **Время:** 1-2 часа на обучение

### Пример диалога

```
AI: Добрый день! Это Анна из компании X. Заметила, что у вас ресторан
    в Сочи — мы помогаем таким бизнесам увеличивать выручку на 20-30%.
    Можно задать пару вопросов?

Лид: Нет времени

AI: Понимаю, у рестораторов каждая минута на счету! Именно поэтому
    предлагаю короткий 15-минутный аудит — бесплатно покажу 3 точки роста
    конкретно для вашего заведения. Когда удобнее — завтра утром или вечером?

Лид: Ну давайте завтра в 10

AI: Отлично! Записала на завтра в 10:00. Скину подтверждение в этот чат.
    Хорошего дня! 🙌

[Передача менеджеру: ТЁПЛЫЙ ЛИД]
```
