## Рекомендации по ценообразованию

### Для партнёров

| Тип лида | Рекомендуемая цена | Аргументация |
|----------|-------------------|--------------|
| Только компания (без телефона) | 10-30₽ | Базовая информация |
| С телефоном карт | 30-50₽ | Общий телефон |
| С телефоном ЛПР | 100-200₽ | Контакт руководителя |
| Квалифицированный лид | 300-500₽ | Проверенный интерес |

### Ценовые модели

1. **За лид** — фиксированная цена за каждый контакт
2. **За базу** — оптовая скидка (100 лидов = 8,000₽ вместо 10,000₽)
3. **Эксклюзив** — лиды только для одного партнёра (x2 цена)
4. **Подписка** — ежемесячная поставка N лидов
