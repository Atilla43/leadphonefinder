# LeadPhoneFinder — Обзор продукта

## Что это?

**LeadPhoneFinder** — инструмент для автоматического сбора B2B-лидов с контактами ЛПР (лиц, принимающих решения). Вы собираете базу компаний с телефонами собственников/директоров и продаёте её партнёрам.

```
"рестораны Сочи" → [Парсинг 2ГИС/Яндекс] → [Пробив телефонов ЛПР] → Excel с контактами → Продажа партнёру
```

---

## Два режима работы

### 1. Поиск компаний по запросу

Вводите запрос — получаете готовую базу:

```
Запрос: "автосервисы Москва"
Результат: 178 компаний с ИНН, адресами, телефонами ЛПР
```

**Возможности:**
- Парсинг 2ГИС и Яндекс.Карт одновременно
- Автоматическое исправление опечаток ("рестараны Сачи" → "рестораны Сочи")
- Дедупликация результатов из разных источников
- Поиск ИНН по названию компании
- Обогащение телефонами собственников

### 2. Загрузка своего файла

Есть файл с ИНН — получите телефоны:

```
Вход: Excel с колонками [ИНН, Название]
Выход: Excel с колонками [ИНН, Название, Телефон, Email, Контакты (ФИО), Адреса, Источники, Записей найдено, Статус]
```

**Возможности:**
- Поддержка Excel (.xlsx) и CSV
- Валидация ИНН по контрольной сумме
- Поиск телефонов через Dyxless API
- Сбор всех данных: телефоны, email, ФИО контактов, адреса, источники
- Статусы: success, not_found, invalid_inn, error

## Бизнес-модель: Продажа лидов

### Схема заработка

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│  Собираете      │ ──▶ │  Формируете     │ ──▶ │  Продаёте       │
│  компании       │     │  базу с ЛПР     │     │  партнёрам      │
└─────────────────┘     └─────────────────┘     └─────────────────┘
```

### Себестоимость vs Продажа

| Метрика | Значение |
|---------|----------|
| **Себестоимость 1 лида** | 0.10-0.50₽ |
| **Рыночная цена лида** | 50-500₽ |
| **Ваша маржа** | 99%+ |

### Кому продавать

| Партнёр | Что нужно | Пример ниши |
|---------|-----------|-------------|
| **Маркетинговые агентства** | Базы для рассылок | Рестораны, салоны красоты |
| **B2B-компании** | Контакты ЛПР | IT-услуги, консалтинг |
| **Call-центры** | Телефоны для обзвона | Банки, страховые |
| **Региональные бизнесы** | Локальные базы | "Все стоматологии Казани" |

## Поддерживаемые данные

### Города (30+)
Москва, Санкт-Петербург, Сочи, Казань, Екатеринбург, Новосибирск, Краснодар, Ростов-на-Дону, Нижний Новгород, Самара, Челябинск, Пермь, Уфа, Волгоград, Красноярск, Омск, Воронеж, Саратов, Тюмень, и другие.

### Категории
- Рестораны, кафе, бары
- Автосервисы, СТО
- Салоны красоты, барбершопы
- Стоматологии, медцентры
- Фитнес-клубы
- Юридические услуги
- IT-компании
- Строительные компании
- И другие

### Источники данных
| Источник | Данные | Доступность |
|----------|--------|-------------|
| **2ГИС** | Название, адрес, телефон, рейтинг, ИНН | Бесплатно |
| **Яндекс.Карты** | Название, адрес, телефон, категория | Бесплатно |
| **DaData** | ИНН по названию компании | 10k/день бесплатно |
| **ЕГРЮЛ** | ИНН (fallback) | Бесплатно |
| **Dyxless API** | Телефоны ЛПР, email, ФИО, адреса | По ключу API |

## Технические требования

### Минимальные
- Python 3.11+
- 2 GB RAM
- 10 GB диск

### Для production
- VPS (Ubuntu 22.04)

## Быстрый старт

### 1. Установка
```bash
git clone <repo>
pip install -r requirements.txt
playwright install chromium
```

### 2. Настройка
```bash
cp .env.example .env
# Заполнить BOT_TOKEN, TELETHON_*, SHERLOCK_*
```

### 3. Запуск
```bash
python -m bot.main
```

### 4. Использование
```
Главное меню → 🔍 Поиск компаний → "рестораны Сочи" → Выбор источника → Выбор лимита (50/100/200/Все) → Обогащение → Excel
```
