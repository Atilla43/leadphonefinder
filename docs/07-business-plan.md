# Бизнес-план LeadPhoneFinder

## 1. Резюме проекта

**LeadPhoneFinder** — платформа автоматизации B2B-продаж полного цикла: от сбора компаний до AI-диалогов с ЛПР.

```
Ввод запроса → Парсинг 2ГИС/Яндекс → Поиск ИНН → Пробив телефонов ЛПР → AI-продажник → Тёплые лиды
```

**Ключевые метрики:**
- Себестоимость лида: 0.10-2.10₽
- Рыночная цена лида: 50-500₽
- Маржинальность: 95-99%
- Окупаемость: 1-й месяц

## 2. Функциональный анализ

### Модуль 1: Скраппер компаний

| Компонент | Функция | Технология |
|-----------|---------|------------|
| QueryParser | Разбор запроса + исправление опечаток | Levenshtein distance, 30+ городов |
| TwoGisScrapper | Парсинг 2ГИС | Playwright headless |
| YandexMapsScrapper | Парсинг Яндекс.Карт | Playwright headless |
| Deduplicator | Удаление дубликатов | Fuzzy matching 85%+ |
| InnFinder | Поиск ИНН компании | DaData API + ЕГРЮЛ |
| ScrapperCache | Кеш результатов 24ч | JSON файлы |

**Результат:** 50-200 компаний с ИНН, адресами, рейтингами за 2-5 минут.

### Модуль 2: Обогащение данных

| Компонент | Функция | Источник |
|-----------|---------|----------|
| SherlockAPIClient | Пробив телефонов ЛПР | Dyxless API (2₽/запрос) |
| SherlockTelegramClient | Fallback пробив | @sherlock_osint_bot (бесплатно) |
| PhoneExtractor | Нормализация телефонов | Regex +7XXXXXXXXXX |
| InnValidator | Валидация ИНН | Контрольная сумма |

**Результат:** Телефоны, email, ФИО руководителей, адреса — 9 колонок в Excel.

### Модуль 3: AI-продажник (Outreach)

| Компонент | Функция | Технология |
|-----------|---------|------------|
| AISalesEngine | Генерация ответов | OpenRouter GPT-4o-mini |
| OutreachService | Рассылка + слушатель | Telethon userbot |
| Автопинг | Напоминания при игноре | Каждые 4ч, 9-21 МСК |
| Детекция тёплых лидов | Уведомление менеджеру | LLM классификация |

**Результат:** Автоматические переговоры с ЛПР → тёплые лиды на блюдечке.

### Модуль 4: Прямая загрузка (новый)

Загрузка Excel с телефонами → сразу в AI-продажник (без скраппинга).

## 5. Финансовая модель

### Unit-экономика одного лида

```
Себестоимость холодного лида:
├─ Парсинг 2ГИС/Яндекс:       0₽ (бесплатно)
├─ Поиск ИНН (DaData):         0.10₽
├─ Пробив телефона (Dyxless):  2.00₽
└─ ИТОГО:                      2.10₽

Себестоимость тёплого лида:
├─ Холодный лид:               2.10₽
├─ AI-диалог (GPT-4o-mini):    ~0.50₽ (3-5 сообщений)
├─ Telethon отправка:          0₽
└─ ИТОГО:                      ~2.60₽

Конверсия холодный → тёплый:   5-10%
Эффективная стоимость тёплого:  2.60₽ / 7.5% = ~35₽
Цена продажи тёплого лида:     500-2,000₽
Маржа на тёплом лиде:          93-98%
```

### Уникальные преимущества (USP)

1. **AI-продажник** — единственный на рынке полный цикл от сбора до AI-переговоров
2. **Тёплые лиды** — не просто контакты, а согласие на встречу/звонок
3. **Телефоны ЛПР** — OSINT пробив, а не только данные ЕГРЮЛ
4. **Себестоимость x100 ниже** — за счёт автоматизации и прямого доступа к API
5. **Telegram-интерфейс** — мобильный, простой, не нужен ПК

```
├─ Доработать AI-продажник по реальным диалогам
```

```
Задачи:

├─ Интеграция с amoCRM/Bitrix24
├─ Fine-tuning GPT на реальных диалогах
```

```
Задачи:

├─ Дашборд аналитики
├─ 100+ клиентов
└─ Выручка: 500,000-2,000,000₽/мес
```

```
Задачи:
├─ Маркетплейс баз (продавцы + покупатели лидов)
├─ Multi-touch (серия сообщений в TG + WhatsApp + email)
```

## 10. Рекомендации по монетизации

### Максимальная прибыль — комбинированная модель

```
Уровень 1: Продажа холодных лидов (100₽/лид)
  └─ Базовый доход, минимальные усилия

Уровень 2: Продажа тёплых лидов (1,000-2,000₽/лид)
  └─ AI-продажник конвертирует 5-10% холодных в тёплые
  └─ Это ГЛАВНЫЙ источник маржи

Уровень 3: SaaS-подписка (3,000-50,000₽/мес)
  └─ Recurring revenue, стабильный доход
  └─ Клиенты сами собирают лиды через ваш бот

Уровень 4: API + White-label
  └─ B2B-партнёры платят за интеграцию
  └─ Масштаб без увеличения команды
```

### Оптимальная стратегия на старте

1. **Начать с продажи тёплых лидов** — максимальная цена, AI-продажник как USP
2. **Целевые ниши:** рестораны, автосервисы, салоны красоты — понятный оффер, массовый рынок
3. **Канал продаж:** Telegram-каналы для бизнеса, партнёрства с маркетинговыми агентствами
4. **Через 3 мес:** запустить SaaS для постоянного дохода
