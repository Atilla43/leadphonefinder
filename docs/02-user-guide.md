# Руководство пользователя

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/scrape` | Поиск компаний по запросу |
| `/help` | Справка |

---

## Сценарий 1: Поиск компаний по запросу

### Шаг 1: Запуск поиска

```
Вы: /scrape

Бот: 🔍 Поиск компаний

     Введите запрос для поиска, например:
     • рестораны Сочи
     • автосервисы Москва
     • салоны красоты Казань

     Или выберите популярный запрос:

     [🍽 Рестораны Москва] [🍽 Рестораны СПб]
     [🚗 Автосервисы Москва] [💇 Салоны Москва]
     [🏥 Стоматологии Москва] [💪 Фитнес Москва]
     [🌴 Рестораны Сочи] [🏨 Отели Сочи]
     [❌ Отмена]
```

### Шаг 2: Ввод запроса

Введите категорию и город. Опечатки исправляются автоматически:

```
Вы: рестараны Сачи

Бот: 📍 Запрос: рестараны Сачи

     ✏️ Исправлено: город: сачи → Сочи, категория: рестараны → рестораны

     📂 Категория: рестораны
     🏙 Город: Сочи

     Выберите источники данных:

     [📍 2ГИС + Яндекс (рекомендуется)]
     [🟢 Только 2ГИС]
     [🔴 Только Яндекс]
     [❌ Отмена]
```

### Шаг 3: Выбор источников

Рекомендуется выбирать оба источника для максимального охвата:

```
Вы: [нажимаете "2ГИС + Яндекс"]

Бот: 🔍 Поиск: рестораны Сочи

     ⏳ Поиск в 2ГИС...
     [▓▓▓▓░░░░░░] 40%
     ━━━━━━━━━━━━━━━━━━━━
     2ГИС: ✅
     Яндекс: ✅
```

### Шаг 4: Результаты поиска

После завершения парсинга вы увидите статистику:

```
Бот: ✅ Поиск завершён!

     📊 Результаты:
     • Найдено: 234
     • Уникальных: 178
     • Дубликатов удалено: 56
     • Из 2ГИС: 156
     • Из Яндекс: 78
     • Время: 142.3 сек

     Хотите обогатить данные телефонами ЛПР?
     Это займёт ~534 секунд.

     [📱 Обогатить телефонами ЛПР]
     [📥 Скачать как есть]
     [❌ Отмена]
```

### Шаг 5: Обогащение телефонами

Выберите "Обогатить телефонами ЛПР" для получения контактов собственников:

```
Вы: [нажимаете "Обогатить телефонами ЛПР"]

Бот: 🔄 Обогащение данных

     Прогресс: 45/178
     Текущая: Ресторан Причал...

     ⏳ Получение телефонов ЛПР...
```

### Шаг 6: Получение результата

После обработки бот отправит Excel-файл:

```
Бот: [📎 leads_20260131_143022.xlsx]

     ✅ Готово!

     📊 Компаний: 178
     📱 С телефонами ЛПР: 89
     📍 С адресами: 178
```

## Сценарий 2: Загрузка файла Excel

### Шаг 1: Подготовка файла

Создайте Excel-файл с двумя колонками:

| ИНН | Название |
|-----|----------|
| 7707083893 | ПАО Сбербанк |
| 7728168971 | ООО Яндекс |
| 2320123456 | ООО Ресторан Причал |

**Требования:**
- Формат: `.xlsx` или `.csv`
- Размер: до 10 МБ
- Строк: до 100

### Шаг 2: Загрузка файла

Отправьте файл в чат с ботом:

```
Вы: [📎 companies.xlsx]

Бот: 📊 Файл загружен

     • Всего строк: 50
     • Валидных ИНН: 47
     • Невалидных ИНН: 3
     • Дубликатов: 0

     [📱 Обогатить телефонами]
     [✅ Только валидация ИНН]
     [❌ Отмена]
```

### Шаг 3: Обработка

```
Вы: [нажимаете "Обогатить телефонами"]

Бот: 🔄 Обогащение данных

     Прогресс: 23/47
     Текущая: ООО Яндекс...

     ⏳ Получение телефонов ЛПР...
```

### Шаг 4: Результат

```
Бот: [📎 enriched_20260131_143500.xlsx]

     ✅ Обогащение завершено!

     📊 Результаты:
     • Обработано: 47
     • Найдено телефонов: 31
     • Не найдено: 16
     • Ошибок: 0
```

## Формат выходных файлов

### При поиске компаний (без обогащения)

| Название | Адрес | ИНН | Телефон (карты) | Рейтинг | Отзывов | Категория | Источник |
|----------|-------|-----|-----------------|---------|---------|-----------|----------|
| Ресторан Причал | ул. Морская, 15 | 2320123456 | +7 862 123-45-67 | 4.5 | 234 | Рестораны | 2gis |

### При обогащении телефонами ЛПР

| ИНН | Название | Адрес | Телефон ЛПР | Телефон (карты) | Статус |
|-----|----------|-------|-------------|-----------------|--------|
| 2320123456 | Ресторан Причал | ул. Морская, 15 | +7 916 123-45-67 | +7 862 123-45-67 | success |

### Статусы

| Статус | Описание |
|--------|----------|
| `success` | Телефон ЛПР найден |
| `not_found` | Телефон не найден в источниках |
| `invalid_inn` | Некорректный ИНН (не прошёл валидацию) |
| `error` | Ошибка при запросе |

## Примеры запросов

### Правильные запросы
```
рестораны Сочи
автосервисы Москва
салоны красоты Казань
стоматологии Екатеринбург
фитнес клубы Санкт-Петербург
it компании Новосибирск
юристы Краснодар
```

### С опечатками (исправятся автоматически)
```
рестараны Сачи → рестораны Сочи
афтосервис Масква → автосервисы Москва
стамотологии Козань → стоматологии Казань
```

### Сокращения городов
```
кафе мск → кафе Москва
бары спб → бары Санкт-Петербург
рестораны питер → рестораны Санкт-Петербург
автосервисы екб → автосервисы Екатеринбург
```

## Часто задаваемые вопросы

### Сколько времени занимает обработка?

| Операция | Время |
|----------|-------|
| Парсинг 100 компаний | 2-5 минут |
| Обогащение 100 компаний | 5-10 минут |
| Общее время на 100 лидов | ~15 минут |

### Какой процент телефонов находится?

Обычно 40-60% компаний имеют найденные телефоны ЛПР. Зависит от:
- Размера компании (крупные чаще)
- Региона (Москва > регионы)
- Категории (B2B > B2C)

### Почему некоторые телефоны не найдены?

- Компания новая (нет данных в OSINT)
- Собственник скрывает контакты
- ИНН некорректный или несуществующий
- Временные ошибки источника

### Можно ли повторить поиск?

Да, просто введите `/scrape` снова. Результаты не кэшируются.

### Есть ли лимиты?

| Параметр | Лимит |
|----------|-------|
| Файл Excel | 100 строк, 10 МБ |
| Результатов с источника | 100 компаний |
| Запросов в день | Без ограничений |

## Советы по использованию

1. **Используйте оба источника** — 2ГИС + Яндекс даёт больше компаний
2. **Проверяйте опечатки** — хотя автоисправление работает, лучше писать правильно
3. **Начинайте с малого** — сначала попробуйте на 10-20 компаниях
4. **Сохраняйте результаты** — файлы не хранятся на сервере
5. **Валидируйте ИНН** — перед загрузкой файла проверьте корректность ИНН
